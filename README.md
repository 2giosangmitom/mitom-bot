# mitom-bot

A personal bot for Discord, built for fun and learning purposes :)

## Features

### Ping Command

Responds with "Pong!" when user uses the `/ping` command.

### Calculate Command

Evaluates simple mathematical expressions provided by the user. For example, if a user inputs `/calculate 2 + 2`, the bot will respond with `2 + 2 = 4`.

The bot supports addition, subtraction, and parentheses. For instance, `/calculate (1 + 2) - 3` will yield `0`.
